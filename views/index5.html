On-Page SEO HTML Elements






1. Title Tag

  • androidClick to Hide

    The title tag specifies the title of the web page, it's displayed in the search engine result pages (SERP). It is important to have a unique web page title because it is the impression of most people if they happen to see your website in the search engine result's page. It doesn't matter if your site is well ranked, a good title is always important to keep the expectations of the person on what he/she is looking for.





2. Meta Description Tag

  • androidClick to Hide

    The meta description tag is a 160 character snippet that summarizes a web page's content, it would let the user know what site he/she is about to click, hence called a description. It is located below the link of the website, it is good to have a good meta description because it would let the user confirm what he/she typed or if it is what the user is looking for.





3. Image

  • androidClick to Hide

    It's a good idea to put a little bit of life in your web page, it would be a great help to the user to understand the content even better, when putting an article it's very important to have an illustration about the article, thus the image is the right element for the job, it makes the page more interesting to read than having none but full of texts packed in to one or more paragraphs which would make it really boring and eye hurting.





4. Text Link

  • androidClick to Hide

    Text links are texts that would direct the user to another website or to another page within the site, the user will be redirected to other page just by clicking, tapping or by hovering. Text links are also called Anchor texts, this would also determine the ranking of the website that would be received in search engines.





5. NoFollowed Link

  • androidClick to Hide

    NoFollow Links are HTML attributes that is used to instruct the search engine bot not to influence a certain hyperlink in the search engine's index, this would prevent and reduce the effectiveness of the certain types of search engine spam.





6. Image Link

  • androidClick to Hide

    Image links are similar to hyperlinks only that when the user hovers the mouse pointer to the image it would be indicated as a clickable image in which it would redirect the user to another site or other parts of the current site the user is using.
