On Page SEO vs Off Page SEO

  • androidClick to Hide

    An On-page SEO is an act of optimizing different parts of the website that affect its search engine results, mostly its more on making the website as optimized as possible in a way that it gets found by the search engine's keywords, and ensuring the contents are what the users are looking for, on the other hand, an Off-page SEO focuses more in the techniques in improving the position of the website in the search engine result's page, it also focuses on increasing the authority of your domain through the act of getting links from other websites.





SEO guides for an On-Page technique

1. Page Titles

  • androidClick to Hide

    The page title is one of the most important SEO factors in a site. Each of the pages & posts have its own unique title, which includes the main keywords for that page.



2. Meta Descriptions

  • androidClick to Hide

    These descriptions are an important place to include relevant keywords for the content, as these are used within the search results when your page is listed.



3. Meta Tags

  • androidClick to Hide

    For each of the pages, it can include a set of keywords in the form of meta tags. These should be all the relevant keywords of the content. which has been researched previously.



4. URL Structure

  • androidClick to Hide

    Including search engine friendly URLs for each of your pages is highly recommended, as these bring better crawling. Shorter URLs seem to perform better in search engine results, however that is not only the factor.



5. Body Tags

  • androidClick to Hide

    When writing the articles, it should break up the content into smaller sections & paragraphs to make it easer for people to read. These sections can be given heading, which is where H1,H2,H3,H4 etc. tags are used.



6. Keyword Density

  • androidClick to Hide

    Including relevant keywords throughout the content is very important, as it helps search engines work out what the content is about. However, it's not a good idea to excessively repeat and overuse keywords just for search engine robots. This can lead to the site being banned from search engines.



7. Image SEO

  • androidClick to Hide

    Using images within the content is a great way to make the site more visually appealing and break up boring chunks of text. A good idea to to utilize these images to help improve the site SEO.

    All the uploaded images have titles, so treat the site the same as the page titles. Including relevant keywords can help people find the site when searcing on Google Images.



8. Internal Linking

  • androidClick to Hide

    Placing links to other website pages, is a great way of improving the site and used properly, internal links can be a useful weapon in the SEO arsenal. Not only does it make it much easier for the visitors to navigate around the site and find all the content, but also it ensures that the site gets properly crawled alowing the search engines to find all of the pages.
